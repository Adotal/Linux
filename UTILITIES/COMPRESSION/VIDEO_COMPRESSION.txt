1. Best Balance (Recommended)

LOW COMPRESSION, MANTAIN H.264, IDENTICAL TO ORIGINAL

$ ffmpeg -i input.mp4 -c:v libx264 -crf 23 -c:a aac -b:a 128k output.mp4

2. Maximum Compression (Smaller File, Slower)

For absolute smallest file size and longer processing times, use the H.265 (HEVC) codec with a slower preset:

$ ffmpeg -i input.mp4 -c:v libx265 -crf 28 -preset slow -c:a aac -b:a 128k output.mp4

PARAMETERS BREAKDOWN
-crf 28		- Number 28 can be 0-51, 51 is the worst quality, smaller size, sane range is 17-28, CFR 30 is quite good tough
-preset		- Encoding speed, slower means best quality and smaller size (smarter compression), values are:
				ultrafast
				superfast
				veryfast
				faster
				fast
				medium – default preset
				slow
				slower
				veryslow

3. Scale Down Resolution (Drastic Size Reduction)

If the video is 4K or 1080p, scaling it down to 720p will slash the file size dramatically:

$ ffmpeg -i input.mp4 -vf scale=-1:720 -c:v libx264 -crf 23 output.mp4

PARAMETERS
i input.mp4: Specifies your source video file.
-c:v libx264 / libx265: Chooses the video encoder. H.264 offers broad compatibility; H.265 compresses much tighter.
-crf 23: Constant Rate Factor controls quality.The scale is 0–51.Lower numbers = higher quality (larger file).Higher numbers = lower quality (smaller file).Range 18–28 is ideal. Every +6 to the CRF number roughly halves the file size.
-preset slow: Tells the encoder how hard to work. Slower presets mean tighter, smarter compression without losing quality. Options include fast, medium, slow, and veryslow.
-c:a aac -b:a 128k: Compresses the audio track to a highly efficient 128 kbps stereo format to save extra space.

Pro-Tips for Compression
Test on a short snippet first: Add -t 10 right after -i input.mp4 to compress just the first 10 seconds as a preview.

Web Streaming: If you are uploading the video to a website, append -movflags faststart to the end of your command. This allows the video to start playing before it finishes downloading entirely


// MAX COMPRESS TO ALL MP4 VIDEOS IN FOLDER, SAVE IN compressed
mkdir -p compressed
for file in *.mp4
do
  echo "Processing: $file"
  ffmpeg -i "$file" -c:v libx265 -crf 30 -preset slow -c:a aac -b:a 128k "compressed/$file"
done
