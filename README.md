# My GNU/Linux

This repo contains my personal text files, personal manuals, installations guides, commands and configurations.

In this repo you can find guides from the most basic commands in linux like ```bash cd, ls, cp, mv, touch,``` to advanced configurations like window-managers and adjusting CPU frequencies (to avoid high temperatures).

Here I have uploaded almost all my learning path using linux distributions, some distros I've used are Ubuntu (naturally my first one), LXQT, Debian, Mint, Linux Lite, MX Linux, Zorin OS, and the most important and the one I use everyday: Arch Linux, yes, I use Arch xD.
