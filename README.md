# My GNU/Linux

This repository contains my personal text files, manuals, installation guides, commands, and configurations. 

Here, you can find guides for everything from the most basic Linux commands (like cd, ls, cp, mv, touch) to advanced configurations, including window managers (dwl, i3, openbox) and CPU frequency adjustments to prevent high temperatures.

I have documented almost my entire Linux learning path. Some of the distros I've used include Ubuntu (naturally my first one), LXQt, Debian, Mint, Linux Lite, MX Linux, Zorin OS, Kali, and the most important one that I use every day: Arch Linux. Yes, I use Arch! xD
